# 01. クイックスタート（15 分で動かす）

～ Windows ユーザーが最短で **Cursor を起動して事務処理プロンプトを実行するまで** の超短縮ルート ～

> 💡 本ファイルは **「動かすこと最優先」のチェックリスト**。各ステップは要点だけ書いています。
> 詰まったときの理由・補足は深追いせず、リンク先（02/03）で確認してください。
> 📌 **ランタイム不要**。Python / Node.js のインストールは要りません。

---

## 全体の流れ（10〜15 分）

```
   ① Cursor をダウンロード & インストール   …… 5 分
   ② サインイン（Hobby = 無料）            …… 2 分
   ③ リポジトリを取得（Clone / ZIP）       …… 3 分
   ④ Cursor で Open Folder                …… 1 分
   ⑤ Chat (Ctrl+L) にプロンプト投入        …… 2 分
        → 🎉 Cursor が事務処理プロンプトに応答する状態
```

---

## チェックリスト

### ① Cursor をインストール
- [ ] https://cursor.com/ →「Download for Windows」で `.exe` を取得し、ダブルクリックでインストール
- 詳細・つまずき → [`02_環境構築ガイド.md`](./02_環境構築ガイド.md) §①②

### ② サインイン（Hobby = 無料）
- [ ] 起動画面の `Sign in` / `Sign up` → メール or Google or GitHub でアカウント作成（**クレカ不要・SMS 認証あり**）
- [ ] 右下にアカウント名が出れば成功
- 安心ポイント（クレカ/SMS/GitHub/データ共有）の解説 → [`02_環境構築ガイド.md`](./02_環境構築ガイド.md) §③

### ③ リポジトリを取得
- [ ] Cursor の `Clone repo` に `https://github.com/kurosawa-kuro/study-cursor.git` を貼る（最も簡単）
- git や ZIP での取得など 3 方法の詳細 → [`03_ハンズオン.md`](./03_ハンズオン.md) §①

### ④ Open Folder
- [ ] `File → Open Folder` で `study-cursor/` を選択 → 「信頼しますか？」は `Yes`
- [ ] 左に `README.md` / `src/office-task/` が見えれば OK

### ⑤ Chat (Ctrl+L) にプロンプト投入
- [ ] `Ctrl+L` で Chat を開き、下のお試しプロンプトを貼って送信:

```
IT エンジニア派遣の営業として、初回商談のお礼メール（件名・本文・署名欄）を日本語で作ってください。
- 宛先: クライアント A 社 採用担当 田中様
- 内容: 本日の Java バックエンドエンジニアの募集ヒアリングのお礼 + 候補エンジニア 1 名を後日提案する旨
```

- [ ] 件名 / 宛名 / 本文 / 署名欄を含む日本語ビジネスメールが返れば成功 🎉
