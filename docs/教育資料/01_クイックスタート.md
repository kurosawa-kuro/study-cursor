# 00. クイックスタート（15 分で Cursor を動かす）

～ Windows ユーザーが最短で **Cursor を起動して事務処理プロンプトを実行するまで** の超短縮ルート ～

> 💡 このドキュメントは **「動かすこと最優先」** で書いています。
> 仕組みの説明や詳細手順は [`02_環境構築ガイド.md`](./02_環境構築ガイド.md) / [`03_ハンズオン.md`](./03_ハンズオン.md) を参照。
> Cursor の使い方詳細（Hobby 枠の制約・Chat/Composer/Tab）は [`04_Cursor操作.md`](./04_Cursor操作.md)。

---

## 全体の流れ

```
   ⏱ 10〜15 分（PC スペックとネットワークによる）

   ① Cursor をダウンロード & インストール  …… 5 分
       ↓
   ② Cursor サインイン（Hobby = 無料）     …… 2 分
       ↓
   ③ リポジトリを取得（clone or DL）       …… 3 分
       ↓
   ④ Cursor で Open Folder                …… 1 分
       ↓
   ⑤ Chat (Ctrl+L) にプロンプト投入        …… 2 分

   → 🎉 Cursor が事務処理プロンプトに応答する状態
```

> 📌 **本キットはランタイム不要**。Python / Node.js のインストールは不要です。

---

## ① Cursor をインストール

1. ブラウザで https://cursor.com/ を開く
2. 「Download for Windows」ボタンから `.exe` インストーラを取得
3. ダウンロードした `Cursor Setup-x.x.x.exe` をダブルクリックしてインストール
4. インストール完了後、Cursor が自動で起動する（しなければスタートメニューから `Cursor` を起動）

---

## ② Cursor にサインイン（Hobby 枠 = 無料）

1. Cursor 起動直後の画面で `Sign in` または `Sign up` をクリック
2. ブラウザが開いて Cursor のサインインページが表示される
3. 既存アカウントがあればサインイン、無ければ無料アカウントを作成
   - 無料アカウント = **Hobby 枠**（月次の slow request 上限あり、詳細は [`04_Cursor操作.md`](./04_Cursor操作.md)）
4. サインインが完了すると Cursor 本体に戻り、右下にアカウント名が表示される

---

## ③ リポジトリを取得

**方法 A（推奨）**: Cursor の Welcome 画面 → `Clone repo` → URL を貼り付け → 保存先を選んで取得

**方法 B**: Windows 標準の `git`（無ければ [git-scm.com](https://git-scm.com/) からインストール）を PowerShell で実行

```powershell
# Windows の cmd / PowerShell で実行（管理者でなくて OK）
mkdir C:\dev
cd C:\dev
git clone https://github.com/kurosawa-kuro/study-cursor.git
```

**方法 C（git すら使いたくない場合）**: GitHub のリポジトリページから `Code → Download ZIP` でダウンロード → 解凍

---

## ④ Cursor で Open Folder

1. Cursor の `File → Open Folder`
2. 取得した `study-cursor/` フォルダを選択
3. 「このフォルダを信頼するか？」と聞かれたら `Yes, I trust the authors`
4. 左サイドバーに `README.md` / `AGENTS.md` / `src/office-task/` が見えれば OK

---

## ⑤ Chat (Ctrl+L) にプロンプトを投入

1. `Ctrl+L` を押す（または右上の Chat アイコンをクリック）
2. 試しに以下を貼って送信（**プロンプト1: 商談後のお礼・提案メール** を最小形で）:

   ```
   IT エンジニア派遣の営業として、初回商談のお礼メール（件名・本文・署名欄）を日本語で作ってください。
   - 宛先: クライアント A 社 採用担当 田中様
   - 内容: 本日の Java バックエンドエンジニアの募集ヒアリングのお礼 + 候補エンジニア 1 名を後日提案する旨
   ```

3. Cursor が以下を返せば成功 🎉:
   - 件名 / 宛名 / 時候の挨拶 / 本文 / 結び / 署名欄 を含む日本語ビジネスメール

慣れたら、次は **プロンプト4: エンジニア一覧 / 案件一覧の集計レポート** で `@file` を使ったデータ読み込みも試してみてください。

---

## 動作確認の早見表

```
  Cursor 起動 → Help → About でバージョン表示
  右下にアカウント名表示                       → サインイン済み
  File → Open Folder で study-cursor を開ける  → リポジトリ準備 OK
  Ctrl+L で Chat が開ける                      → プロンプト投入可能
  上記 ⑤ のプロンプトに応答が返る              → Hobby 枠で動作確認 OK
```
