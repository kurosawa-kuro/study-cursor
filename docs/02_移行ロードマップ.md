## 1. プロンプトメモ（`src/office-task/`）の追加候補

> 既実装: ファイルリネーム / Excel 複数集計 / PDF テキスト抽出 / メール文面生成 / 議事録テンプレ / ファイル整理 / CSV 集計レポート

| 候補 | 概要 | 優先度 | 実現手段（ランタイム非インストール前提） |
|---|---|---|---|
| ディレクトリ重複ファイル検出 | ハッシュベースで重複を一覧化 | 低 | PowerShell `Get-FileHash` を Cursor が生成 |
| Excel → PDF 変換 | Excel のエクスポート機能を案内 | 低 | Excel の「名前を付けて保存」 → PDF。Cursor が手順書を提示 |
| Markdown 議事録 → メール送信用テキスト整形 | 既存の議事録テンプレと連携 | 低 | Cursor Chat 内のテキスト変換のみ |
| 画像ファイルの一括リネーム（撮影日時付与） | 写真のメタ情報から日付を取り出す | 低 | PowerShell の `Get-ItemProperty` で対応可能 |
